Detailed Description

LAPACK wrappers for double precision real array.

Implemented in terms of Row-Major array as C/C++ style Since they contain transposition of array before and after calling LAPACK subroutines, performance is not good as FORTRAN

Or, if you work with Col-Major array, use clapack_xxx subroutines directly (see lapack/clapack.h)
