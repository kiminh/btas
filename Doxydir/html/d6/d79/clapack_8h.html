enum CLAPACK_TRANSPOSE

Transposition type used in CLAPACK.

This is a part of BTAS library to provide C/C++ interface of LAPACK calls. Note that all functions work with Column-Major array with respect to FORTRAN-style, since C-style Row-Major array is not supported in LAPACK library.

Wrapper functions for Row-Major array are provided in BTAS dense array layer.
