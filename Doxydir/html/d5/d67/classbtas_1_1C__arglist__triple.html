Detailed Description

template<typename T, size_t N1, size_t N2, size_t N3>
class btas::C_arglist_triple< T, N1, N2, N3 >

Argment list for contraction.

i.e. BLAS (level 2, 3): GEMV, GER, GEMM, etc. m_arglist has pairs of array (a, b) and m_c_ptr has the pointer to array (c), for function call s.t. c = sum_{i} foo(a[i], b[i])
