Detailed Description

Utility functions for fixed and variable size vectors.

Boost array or STL array (C++11) class is used as a fixed size vector TVector Convenient shaping function, dot product, sequence constructor, transposition, and permutation functions are provided.

C/C++ interfaces of fast copying, scaling, and adding are provided, which are specialized for each variable type to call corresponding BLAS function.
